About

This website is made for people who would like to learn more about how to make their own hand craft projects by crocheting and or knitting. It can be very intimidating to start but once you know the basics, it will definitely be easier to do the more complex items. For crochet, it isn't advisable for you to start from plushies since they are a little bit more advanced but once you know how to do the simple stitches you can start right away! For knitting on the other hand, you should start with something square or rectangular shaped instead of something in the round so you can practice casting on, the stitches, and casting off. While you can learn how to do the stitches as you go, it's going to make the end result of your project not look the best it could be.



FaQ


1. What is the Difference Between Crochet & Knitting

While you can make pretty much the same things using either, there are some differences like the tools that are used and the different kinds of stitches that will result the items looking different from each other.


2. Is Knitting Easier Than Crocheting?

At the end it depends on the person and the project you are doing so it is hard to say which is easier but considering that knitting needs to use two needles it will test your hand dexterity comparing to crochet which will only use one hook. There are benefits on learning both like if a stitch drops while you are knitting, it can be very helpful to have a crochet hook to easily fix it.


3. Is Crocheting Faster Than Knitting?

In general, techincally crochet is faster because of the larger stictches but it does use more yarn, on the other hand knitting can take a long time and patience since the stitches are smaller so it will take longer to build up but it would use up less yarn overall.
